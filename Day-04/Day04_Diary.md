# Day 04 – CSS Flexbox and Media Queries

**Date:** 26 June 2026

## Objective

The objective of today's session was to understand the CSS Flexbox layout model and learn how Media Queries can be used to create responsive web pages that adapt to different screen sizes.

## Topics Covered

During today's training session, I was introduced to the Flexbox layout model, which provides an efficient way to arrange and align elements within a container. The following concepts were covered:

* Introduction to CSS Flexbox
* Flex container and flex items
* Flex direction
* Justify-content
* Align-items
* Flex-wrap
* Gap property
* Basic responsive web design
* Introduction to Media Queries
* Changing the layout based on different screen sizes

## Task Assigned

The instructor provided a reference image of a webpage layout and assigned the task of recreating the layout using CSS Flexbox. In addition, a Media Query was to be implemented so that the layout responded appropriately when the browser window or screen size changed.

## Work Completed

I recreated the given layout using the Flexbox layout model by arranging elements according to the reference design. I also implemented a Media Query to modify the layout for smaller screen sizes and tested the responsiveness by resizing the browser window. The program was successfully executed and verified in Visual Studio Code and a web browser.

## Learning Outcomes

By the end of the session, I understood how Flexbox simplifies the process of creating flexible and well-aligned layouts. I also learned the importance of responsive web design and how Media Queries help websites adapt to different devices, improving usability and user experience.

## Conclusion

Today's session introduced important CSS concepts that are widely used in modern web development. The practical assignment helped me gain hands-on experience in building responsive layouts using Flexbox and Media Queries, strengthening my understanding of front-end design principles.
