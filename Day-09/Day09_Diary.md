# Day 09 – JavaScript Functions

**Date:** 03 July 2026

## Objective

The objective of today's session was to understand the concept of functions in JavaScript and learn how they are used to organize code into reusable blocks. The session also introduced parameters, arguments, and arrow functions, which provide a modern and concise syntax for writing functions.

## Topics Covered

During today's training session, I learned about JavaScript functions and their importance in simplifying program development. The following concepts were discussed:

* Introduction to functions
* Function declaration and function calling
* Function syntax
* Parameters and arguments
* Return statement
* Functions with and without parameters
* Functions with and without return values
* Arrow functions
* Advantages of using functions for code reusability and modularity

## Practical Work

To reinforce the concepts covered during the session, I practiced writing JavaScript programs using different types of functions. The programs included functions with parameters and arguments, functions that returned values, and arrow functions for performing various operations. The programs were successfully executed and tested in Visual Studio Code.

## Learning Outcomes

By the end of the session, I understood how functions help divide a program into smaller, reusable units, making the code easier to read, maintain, and debug. I also learned the difference between parameters and arguments, the purpose of the `return` statement, and how arrow functions provide a simpler syntax for defining functions in JavaScript.

## Conclusion

Today's session introduced one of the most fundamental concepts of programming. Learning about functions enhanced my understanding of code organization and reusability, while practicing different types of functions strengthened my programming skills and prepared me for more advanced JavaScript concepts in the upcoming sessions.
