# Day 10 – JavaScript Strings and String Methods

**Date:** 02 July 2026

## Objective

The objective of today's session was to understand the concept of strings in JavaScript and learn how built-in string methods can be used to manipulate, search, modify, and process textual data efficiently.

## Topics Covered

During today's training session, I learned about strings, one of the most commonly used data types in JavaScript. The following concepts were discussed:

* Introduction to strings
* String declaration and initialization
* Accessing characters using indexing
* String length
* Common string methods, including:

  * `charAt()`
  * `indexOf()`
  * `includes()`
  * `slice()`
  * `substring()`
  * `replace()`
  * `replaceAll()`
  * `split()`
  * `trim()`
  * `toUpperCase()`
  * `toLowerCase()`
  * `startsWith()`
  * `endsWith()`
  * `repeat()`
  * `padStart()`
  * `padEnd()`

## Practical Work

To reinforce the concepts covered during the session, I practiced writing JavaScript programs demonstrating the use of various string methods. The programs involved performing operations such as searching for characters or words, extracting substrings, replacing text, converting letter cases, trimming unnecessary spaces, splitting strings into arrays, and formatting string output using different built-in methods. All programs were successfully executed and tested in Visual Studio Code.

## Learning Outcomes

By the end of the session, I gained a clear understanding of how strings are stored and manipulated in JavaScript. I learned how different string methods simplify common text-processing tasks and improve code efficiency. Through practical exercises, I became familiar with selecting appropriate methods based on different programming requirements.

## Conclusion

Today's session provided valuable insight into string manipulation, an essential aspect of JavaScript programming. The practical implementation of various string methods strengthened my understanding of text processing and prepared me for handling string-related operations in real-world web development projects.
