# Day 02 – HTML Tables and Forms

**Date:** 24 June 2026

## Objective

The objective of today's session was to understand how HTML tables and forms are used to organize data and collect user input on web pages. The session focused on the structure, elements, and practical implementation of tables and forms.

## Topics Covered

During today's training session, I learned about HTML tables and forms, which are essential components of web development.

The following concepts were discussed:

* Structure of an HTML table
* Table rows, headers, and data cells using `tr`, `th`, and `td`
* Table borders and formatting
* Merging cells using `rowspan` and `colspan`
* Form creation using the `form` tag
* Input fields for collecting user information
* Labels and placeholders
* Radio buttons and checkboxes
* Drop-down lists and text areas
* Submit and reset buttons
* Organizing form elements for better user interaction

## Task Assigned

The task assigned for the day was to recreate sample tables and forms provided by the instructor. The objective was to understand how different HTML elements work together to create structured layouts and user input interfaces.

## Work Completed

To practice the concepts covered in the session, I created multiple HTML files demonstrating:

* Different table structures using rows, columns, and headers
* Usage of `rowspan` and `colspan` for complex table layouts
* Registration and information forms using various input elements
* Form layouts based on reference images provided by the instructor
* Proper organization and alignment of form components

All files were successfully executed and tested in Visual Studio Code and a web browser.

## Learning Outcomes

By the end of the session, I gained practical experience in designing tables and forms using HTML. I learned how tables can be used to present structured data and how forms enable user interaction by collecting information through various input controls. I also understood the importance of proper layout and organization in creating user-friendly web pages.

## Conclusion

Today's session strengthened my understanding of HTML by introducing interactive and data-oriented elements. The practical exercises of recreating tables and forms helped improve my coding skills and provided hands-on experience in building common web page components used in real-world applications.
