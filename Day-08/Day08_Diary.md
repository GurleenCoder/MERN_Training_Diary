# Day 08 – Conditional Statements and Loops in JavaScript

**Date:** 02 July 2026

## Objective

The objective of today's session was to understand how JavaScript controls the flow of program execution using conditional statements and loops. These concepts enable programs to make decisions and perform repetitive tasks efficiently.

## Topics Covered

During today's training session, I learned the fundamental control structures of JavaScript that are used to implement program logic. The following concepts were discussed:

* Introduction to conditional statements
* `if` statement
* `if...else` statement
* `if...else if...else` ladder
* Nested `if` statements
* `switch` statement
* Introduction to loops
* `for` loop
* `while` loop
* `do...while` loop
* Basic use of `break` and `continue` statements

## Practical Work

To reinforce the concepts covered during the session, I practiced writing JavaScript programs that demonstrated the use of conditional statements and loops. The practice included solving simple logical problems, making decisions based on different conditions, and generating repetitive outputs using various looping constructs. All programs were successfully executed and tested in Visual Studio Code.

## Learning Outcomes

By the end of the session, I gained a clear understanding of how conditional statements help a program make decisions based on different situations. I also learned how loops reduce code repetition by executing a block of code multiple times until a specified condition is met. Through practice, I became familiar with selecting appropriate control structures based on different programming requirements.

## Conclusion

Today's session introduced the core programming concepts of decision-making and iteration, which are essential for developing logical and efficient JavaScript programs. The practical exercises strengthened my problem-solving skills and provided a strong foundation for learning more advanced programming concepts in the upcoming sessions.
