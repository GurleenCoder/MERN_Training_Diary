# Day 06 – CSS Grid Layout

**Date:** 30 June 2026

## Objective

The objective of today's session was to understand the CSS Grid Layout system and learn how it can be used to create structured and responsive web page layouts by arranging elements in rows and columns.

## Topics Covered

During today's training session, I was introduced to the CSS Grid Layout, which is a two-dimensional layout system used for designing complex web page structures. The following concepts were covered:

* Introduction to CSS Grid
* Grid container and grid items
* Grid rows and columns
* `grid-template-rows` and `grid-template-columns`
* The `repeat()` function
* Row and column gaps
* Grid item placement
* Creating structured webpage layouts using CSS Grid

## Task Assigned

The instructor provided a reference grid layout and assigned the task of recreating it using CSS Grid. The objective was to accurately arrange the elements according to the given design while applying the concepts learned during the session.

## Work Completed

I successfully recreated the reference grid layout using CSS Grid properties. The layout was designed by defining appropriate rows and columns, positioning grid items correctly, and maintaining proper spacing between elements. The program was tested in Visual Studio Code and verified in a web browser to ensure that it matched the reference layout.

## Learning Outcomes

By the end of the session, I developed a clear understanding of how CSS Grid simplifies the process of designing structured web layouts. I learned how to organize elements efficiently using rows and columns and understood the difference between one-dimensional layouts created with Flexbox and two-dimensional layouts created with CSS Grid.

## Conclusion

Today's session enhanced my understanding of modern CSS layout techniques. Recreating the reference grid provided valuable hands-on experience and improved my ability to build organized and visually structured web page layouts using CSS Grid.
