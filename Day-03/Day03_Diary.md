# Day 03 – Introduction to CSS

**Date:** 25 June 2026

## Objective

The objective of today's session was to understand the fundamentals of Cascading Style Sheets (CSS) and learn how CSS is used to control the appearance, layout, and presentation of web pages.

## Topics Covered

During today's session, I was introduced to the basic concepts of CSS and its role in web development. The following topics were covered:

* Introduction to CSS
* Types of CSS (Inline, Internal, and External)
* CSS Selectors (Element, ID, Class, and Group Selectors)
* Typography properties such as font family, font size, font style, font weight, and text color
* Background colors and border styling
* CSS Box Model (Content, Padding, Border, and Margin)
* Box sizing (`box-sizing`)
* Display properties with emphasis on block and inline elements

## Practical Work

Although no specific coding assignment was given during the session, I independently practiced the concepts covered in class by creating HTML pages styled with CSS. The practice focused on:

* Applying different types of selectors
* Styling text using typography-related properties
* Experimenting with colors, borders, margins, and padding
* Understanding the CSS Box Model through practical examples
* Demonstrating the difference between block and inline elements using the `display` property

The programs were successfully executed and tested in Visual Studio Code and a web browser.

## Learning Outcomes

By the end of the session, I developed a clear understanding of how CSS enhances the appearance of HTML webpages. I learned how different selectors target specific elements, how the Box Model controls spacing and layout, and how typography and display properties improve the visual presentation of a webpage.

## Conclusion

Today's session introduced the fundamental concepts of CSS that form the basis of modern web design. The self-practice exercises helped reinforce my understanding and improved my confidence in styling web pages effectively.
