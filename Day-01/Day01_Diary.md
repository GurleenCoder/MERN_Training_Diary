# Day 01 – Introduction to HTML and Web Page Structure

**Date:** 23 June 2026

## Objective

The objective of today's session was to gain a basic understanding of HTML and learn how different tags are used to structure and display content on a web page.

## Topics Covered

During the first day of training, I was introduced to the fundamentals of HTML (HyperText Markup Language), which is the standard language used for creating web pages. The session covered various categories of HTML tags and their applications.

The following concepts were discussed:

* Different levels of headings (`h1` to `h6`)
* Paragraph and line break tags
* Semantic and non-semantic HTML elements
* Block-level and inline elements
* Text formatting tags such as bold, italic, underline, strong, emphasis, superscript and subscript
* Ordered and unordered lists
* Anchor tags for hyperlinks
* Image tag and favicon
* Audio and video tags
* Iframe tag for embedding external content
* Structural tags including `header`, `nav`, `main`, `section`, `article`, `aside`, and `footer`

## Task Assigned

The task assigned for the day was to create practice web pages demonstrating the use of commonly used HTML tags and page-structuring elements. The objective was to understand how different tags affect the layout and presentation of content on a webpage.

## Work Completed

To reinforce the concepts learned during the session, I created multiple HTML practice files covering:

* Basic text formatting and heading tags
* Lists and hyperlinks
* Semantic page structure using modern HTML elements
* Image embedding, favicon implementation, iframe usage, and demonstration of block and inline elements

All files were successfully executed in Visual Studio Code and tested in a web browser.

## Learning Outcomes

By the end of the session, I gained a clear understanding of how HTML is used to create the structure of a webpage. I learned the importance of semantic tags in improving readability and organization of web content. I also understood the difference between block-level and inline elements and practiced embedding images and external content into web pages.

## Conclusion

The first day provided a strong foundation for web development by introducing the essential building blocks of HTML. The practical exercises helped in understanding the purpose and usage of various tags, preparing me for more advanced topics in the upcoming sessions.
